== Specification Scope

This product specification outlines the development of data from inception to the end user, through an authoritative weather forecasting agency. Requirements for data and metadata are provided. This document does not include product delivery mechanisms.

*Scope Identification*:: Global
*Hierarchical Level*:: 006 - series
*Hierarchical Level Name*:: Marine Weather Warnings Dataset

*Extent*:: EX_GeographicExtent - Global coverage of maritime areas. +
EX_TemporalExtent - Not defined for this product specification. +
EX_VerticalExtent - Not defined for this product specification.
